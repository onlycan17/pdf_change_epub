# PDF to EPUB 변환기 TODO 리스트

## 프로젝트 개요
Python 기반 PDF → EPUB 변환기로, 한국어 처리에 최적화되고 React 기반 웹 프론트엔드를 제공합니다. OCR 기술을 활용한 스캔 문서 지원, EPUB3 표준 준수, CLI/API 인터페이스를 목표로 합니다.
