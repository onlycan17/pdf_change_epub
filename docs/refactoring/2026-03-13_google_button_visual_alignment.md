# 로그인 화면 Google 버튼 시각 정렬

## 배경

- 로그인 화면의 기본 버튼들은 같은 폭과 높이 규격을 사용하지만 Google 로그인 버튼은 Google SDK가 직접 렌더링해 크기가 다르게 보인다.
- 사용자는 소셜 로그인 버튼도 일반 버튼과 같은 시각 리듬을 갖도록 정렬된 화면을 원한다.

## 목표

- Google 로그인 버튼이 일반 버튼과 비슷한 폭과 높이로 보이게 만든다.
- SDK 동작은 유지하고, 레이아웃 껍데기만 정돈한다.
- 로그인/회원가입 화면에서 같은 버튼 규격을 재사용할 수 있게 한다.

## 비목표

- Google 로그인 방식 자체를 바꾸지 않는다.
- 로그인 폼의 정보 구조나 텍스트는 바꾸지 않는다.

## 변경 전략

1. Google 버튼을 감싸는 외부 래퍼를 추가해 폭과 높이 기준을 고정한다.
2. SDK가 그리는 버튼은 그 래퍼 안에서 꽉 차게 보이도록 맞춘다.
3. 비활성화 상태와 오류 메시지는 기존 동작을 유지한다.

## 검증 계획

- `npm run format`
- `npm run lint`
- `npm run build`
- Playwright로 로그인 화면을 열어 버튼 정렬 상태를 확인한다.
