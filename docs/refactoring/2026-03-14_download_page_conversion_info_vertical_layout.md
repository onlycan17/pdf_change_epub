# 다운로드 화면 변환 정보 세로 레이아웃 정리

## 배경

- 다운로드 화면의 `변환 정보` 영역이 가로 2열로 배치되어 긴 파일명과 수치 정보가 한 줄에 몰려 보인다.
- 사용자는 항목을 위에서 아래로 읽을 수 있는 세로 구조를 원한다.

## 목표

- `변환 정보`를 항목별 세로 카드 목록으로 바꿔 한 번에 한 정보씩 읽히게 만든다.
- 기존 다운로드 동작과 안내 문구는 유지한다.
- 테스트로 핵심 정보 항목이 모두 노출되는지 확인한다.

## 비목표

- 다운로드 API 호출 방식은 바꾸지 않는다.
- 상단 성공 배지와 광고 영역 구조는 바꾸지 않는다.

## 변경 전략

1. `DownloadPage`에서 변환 정보 표시 데이터를 배열로 정리한다.
2. 2열 그리드 대신 세로 목록 카드로 렌더링해 정보 간 간격과 구분을 명확하게 만든다.
3. 페이지 테스트를 추가해 주요 정보 항목과 값이 표시되는지 검증한다.

## 검증 계획

- `npm run test`
- `npm run lint`
- `npm run format`
- `npm run type-check`
