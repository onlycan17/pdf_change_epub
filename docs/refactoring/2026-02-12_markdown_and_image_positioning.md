# 2026-02-12 마크다운 구조화 및 이미지 원위치 반영

## [배경]
- 현재 변환 로직은 마크다운 텍스트를 줄 단위 `<p>`로 감싸는 방식이라, 제목/목록/코드블록 등 문서 구조가 EPUB에 올바르게 반영되지 않습니다.
- 이미지도 본문 끝 갤러리로 붙는 방식이라, 원문 중간 위치 맥락이 깨집니다.

## [계획]
- PDF 추출 단계에서 페이지별 콘텐츠 흐름(텍스트 블록/이미지 블록 순서)을 함께 수집합니다.
- 오케스트레이터에서 이미지 자산을 페이지/순서 기반으로 매핑하고, 페이지 본문 중 해당 위치에 `<figure><img/></figure>`를 삽입합니다.
- 마크다운은 단순 줄바꿈 변환 대신 구조화 렌더러를 통해 EPUB용 XHTML로 변환합니다.
  - heading, list, blockquote, code fence, paragraph 지원
- 기존 갤러리 덧붙이기 방식을 제거하고 본문 중심 배치로 변경합니다.

## [영향도]
- EPUB 결과 품질(구조/가독성/원문 맥락)이 향상됩니다.
- 이미지 배치가 페이지 내 실제 흐름을 우선 반영하게 됩니다.
- API 스펙 변경 없이 내부 변환 품질만 개선됩니다.
