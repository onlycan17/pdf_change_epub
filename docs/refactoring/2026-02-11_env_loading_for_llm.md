# 2026-02-11 백엔드 .env 로딩 및 LLM 설정 반영 보강

## [배경]
- 사용자가 루트 `.env`에 `OPENROUTER_API_KEY`를 설정했지만, 실행 중 설정 객체에서 값이 비어있는 상태가 확인되었습니다.
- 원인: 현재 설정 클래스가 OS 환경변수만 읽고 `.env` 파일을 자동 로딩하지 않는 구조입니다.

## [계획]
- Pydantic Settings의 `env_file` 옵션을 적용해 루트 `.env`를 자동 로딩합니다.
- `LLM_BASE_URL`이 빈 문자열일 때 기본 OpenRouter URL로 되돌리는 안전 로직을 추가합니다.
- 문맥 보정 시 어떤 모델이 실제 호출되었는지 로그를 남겨 진단 가능성을 높입니다.

## [영향도]
- 기존 API/기능 인터페이스 변경은 없습니다.
- `.env`에 작성한 키가 즉시 설정으로 반영되어 LLM 호출이 정상화됩니다.
- 운영/개발에서 설정 추적이 쉬워집니다.
