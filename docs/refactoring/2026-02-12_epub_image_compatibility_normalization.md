# 2026-02-12 EPUB 이미지 호환성 정규화

## [배경]
- 일부 PDF는 JPX/JBIG2 등 EPUB 리더가 바로 렌더링하지 못하는 이미지 포맷을 포함합니다.
- 현재 로직은 포맷명이 미지원일 때 확장자만 PNG로 바꾸고 원본 바이트를 유지할 수 있어, 뷰어에서 이미지가 보이지 않는 문제가 발생할 수 있습니다.

## [계획]
- 변환 오케스트레이터에서 추출 이미지를 EPUB 호환 포맷으로 정규화합니다.
  - 지원 포맷: png/jpg/jpeg/gif/webp/bmp
  - 미지원 포맷: PIL로 PNG로 재인코딩
  - 재인코딩 실패: 해당 이미지만 건너뛰고 변환은 계속
- 단위 테스트를 보강해 미지원 포맷 입력 시 PNG 변환이 적용되는지 검증합니다.

## [영향도]
- EPUB 내 이미지 표시 안정성 향상
- 이미지 포맷 변환 비용으로 처리 시간이 소폭 증가할 수 있음
- API 인터페이스 변화 없음
